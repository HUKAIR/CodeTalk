- **护城河** = 零-LLM 确定性接地 / 逐字引真实 SHA·测试·PR 原话,对抗 AI 反推式编造。**本地优先只是被动加分项**(已被 Pieces 抹平),不是核心、不上首屏当购买理由。

- **分发** = 寄生(MCP 已发 / IDE 扩展待验证),进用户已在用的工具,**不做独立 app**。
- **变现** = 软件**之外**的东西(商用授权 / 合规背书 / 价值兑现凭证),不是软件运行权;付费墙在时序闸 G1/G2/G3 同绿前 NEVER 启。
- **隐私** = 已是三档梯度(`no_llm`/`ollama` 零-egress → 默认云 LLM 脱敏出网 → 本地大模型),红线真实边界 = 「LLM 例外 + 脱敏」;**架构级零-egress 是 air-gapped 合规付费支点,不可降级为承诺**。
